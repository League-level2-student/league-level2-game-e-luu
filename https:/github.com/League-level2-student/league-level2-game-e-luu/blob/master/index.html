Flappy Bird

by: Emmy Lii

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because I really liked it when I was younger. I get a lot of nostalgia whenever I play it :)

2. What I Learned

I learned how to make my game user friendly. Some feedback I got was that the bird jumped too far every time you clicked so I changed that to a smaller height.

3. Architecture of My Game

A pipe moves from right to left and the bird stays in one spot and only moves on its y-axis. If the bird hits the pipe, you lose and the game ends.

4. Technical Implementation

I am proud that I got the if statement to work that stated that if the bird was in the pipe, the game ended. I first completed the code for both the pipe and bird then used trial and error to determine how I was suppose to do my if statement.

5. Most Challenging Part

I most challenging thing was probably getting the pipe to work. It was very difficult to create pipes of different heights that moved from right to left but eventually I figured out a solution to this by making one pipe go from left to right then resetting again on the right.

6. Future Fixes and Updates

I would like to add an image to every pipe and I would like to change the menu and end screens.
